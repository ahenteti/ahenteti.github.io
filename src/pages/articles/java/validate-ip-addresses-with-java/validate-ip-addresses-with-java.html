Using java Regex

Code

            
              public boolean isValidIpv4Address(String ipAddress) {
                  String zeroTo255Pattern = "(\\d{1,2}|[0-1]\\d{2}|2[0-4]\\d|25[0-5])";
                  String ipv4AddressPattern = zeroTo255Pattern + "\\." + zeroTo255Pattern + "\\." + zeroTo255Pattern + "\\." + zeroTo255Pattern;
                  return ipAddress.matches(ipv4AddressPattern);
              }

Explanation

  • \\d{1,2} matches one or two digits
  • [0-1]\\d{2} matches three digits starting with 0 or 1
  • 2[0-4]\\d matches numbers between 200 and 249
  • 25[0-5] matches numbers between 250 and 255

Using InetAddress.getByName method

The InetAddress.getByName method treats also IPv6 addresses. For me, we should use this solution for production code 

            
              public boolean isValidIpAddress(String ipAddress) {
                  try {
                      InetAddress.getByName(ipAddress);
                      return true;
                  } catch (UnknownHostException e) {
                      return false;
                  }
              }
