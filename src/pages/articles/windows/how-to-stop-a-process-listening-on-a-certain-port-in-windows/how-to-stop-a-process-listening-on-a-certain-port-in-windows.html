Find the process id from the server port

netstat -a -b -o | findstr server-port

Kill the process

taskkill /F /PID process-id

Output example

                    » netstat -a -b -o | findstr 60071
                      TCP   0.0.0.0:60071   LISTENING   4264

                    » taskkill /F /PID 4264
