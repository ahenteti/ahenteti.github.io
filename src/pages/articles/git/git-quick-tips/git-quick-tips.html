How to get git history

git log --oneline --decorate --graph --all

How to get unstaged changes

git diff

How to get staged changes

git diff --cached

How to get git aliases

git config -l | grep alias

How to show changed files for a commit

git show --pretty= --name-only

How to get current branch

git branch | sed -n -e 's/^\\* \\(.*\)/\\1/p'

How to add aliases to git

git config --global alias.glog 'log --oneline --decorate --all --graph'

How to amend a commit without changing last commit message

git commit -a --amend --no-edit
