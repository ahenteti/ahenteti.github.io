How to convert a string to lower case

          echo 'HELLO WORLD!' | tr '[:upper:]' '[:lower:]'

How to uppercase the first character in a variable

          foo=\$(tr '[:lower:]' '[:upper:]' <<< \${foo:0:1})\${foo:1}

How to split a string on delimiter

          IFS='-' read -ra array <<< "a-b-c"; unset IFS
          for i in \${array[@]}; do
              echo \$i
          done

How to remove particular characters from a string

          # using tr
          echo "Hello World!" | tr -d !

          # using sed
          echo "Hello World!" | sed 's/!//g'

How to prevent read command to eat backslashes

you should use the -r option of the read command

          read -r path

How to extract filename and extension in Bash

          filename=\$(basename "\$fullfile")
          extension="\${filename##*.}"
          filename="\${filename%.*}"
