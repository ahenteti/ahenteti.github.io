Before adding React

I was already using webpack and bable to bundle and transpile my project and here are the steps that I have followed to add React to this project to discover its components which are more and more popular today ;)

Add react & react-dom dependencies

Add @babel/preset-react preset dependency

Update .babelrc configuration file

            
              {
                "presets": ["@babel/preset-env", "@babel/preset-react"]
              }

After these changes

I was able to add my first React component to my website



Result
react component code 
              
                import * as React from 'react';
                
                const HelloReact = () => <button>Hello React<button>;

                export default HelloReact;
code added in the current html page 
              
                <div id="first-react-component"></div>
code added in the current javascript page 
              
                import * as React from 'react';
                import * as ReactDOM from 'react-dom';
                import HelloReact from './hello-reactcomponent';

                const reactComponentContainer = document.getElementById('first-react-component');
                ReactDOM.render(<HelloReact />, reactComponentContainer);
