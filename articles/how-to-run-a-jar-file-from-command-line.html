Jars with predefined main class

Every jar must contain a MANIFEST.MF file under the META-INF folder which holds useful information about the jar itself and eventually the main class present inside it. Here is an example of MANIFEST.MF with Main-Class property set on it 

          
            Manifest-Version: 1.0
            Archiver-Version: Plexus Archiver
            Built-By: ahenteti
            Created-By: Apache Maven 3.6.2
            Build-Jdk: 1.8.0_221
            Main-Class: io.ahenteti.ahenteti.java.App

Jars with dependencies

Suppose that you have a folder lib which exists inside the project and holds third-party libraries used by your application and your jar wouldn't work without including these dependencies

Option 1 : Specify the dependencies while running the jar in the command line

Option 2 : Specify the classpath of the jar in the MANIFEST.MF file

          
            Manifest-Version: 1.0
            Archiver-Version: Plexus Archiver
            Built-By: ahenteti
            Class-Path: libs/*
            Created-By: Apache Maven 3.6.2
            Build-Jdk: 1.8.0_221
            Main-Class: io.ahenteti.ahenteti.java.App
